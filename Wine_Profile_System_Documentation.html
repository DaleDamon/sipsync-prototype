Complete V5 Scoring Matrix

Scoring System Overview

The V5 scoring matrix uses variable point values (1, 2, or 3 points) to maximize differentiation between profiles and reduce tie frequency. This approach increases the number of possible score outcomes from 7 (in uniform 2-point systems) to 30+, creating more nuanced profile assignments.

Point Value Meanings:

  • 3 Points: Strong correlation - answer is highly predictive of the profile
  • 2 Points: Medium correlation - standard answer-to-profile mapping
  • 1 Point: Weak correlation - answer loosely suggests the profile

Polar Graph Specifications

5-Dimensional Wine Preference Model

Each wine profile is represented in a 5-dimensional radar graph showing numeric values (0-100 scale) for five core wine characteristics. These dimensions define each profile's unique "flavor fingerprint" that determines which wines they will enjoy most.

The 5 Wine Dimensions:

  • Acidity: Brightness and tartness (Low 0-50 ‚Üí High 50-100)
  • Tannins: Drying sensation in mouth (Low 0-50 ‚Üí High 50-100)
  • Body Weight: Fullness on palate (Light 0-50 ‚Üí Full 50-100)
  • Sweetness: Residual sugar level (Dry 0-50 ‚Üí Sweet 50-100)
  • Flavor Intensity: Overall boldness (Subtle 0-50 ‚Üí Bold 50-100)

Quiz Answer ‚Üí Wine Dimension Mapping

How the Quiz Determines Dimension Values

The V5 quiz does not directly measure wine dimensions. Instead, it measures preference for everyday foods and drinks that correlate with wine characteristics. Here's how the process works:

The Process Flow:

  1. User completes 15 questions about everyday food and drink preferences
  2. Each answer awards points to one wine profile (using the V5 scoring matrix)
  3. Backend calculates total points for all 10 profiles
  4. The profile with the highest score wins (ties broken randomly for fairness)
  5. The winning profile's predefined dimension values are assigned to the user
  6. The polar graph renders the user's 5-dimensional preference profile

Which Quiz Questions Influence Each Dimension?

Acidity (Brightness/Tartness)

Questions that directly or indirectly measure preference for acidic, bright, tart flavors:

  • Q4: Sour/Tangy Flavors - Most direct measurement of acidity preference
  • Q5: Salad Dressing - Vinaigrette dressing indicates acidic preference
  • Q6: Morning Beverage - Citrus juice indicates preference for bright acidity
  • Q14: Fruit Type - Citrus fruits correlate with acidic wine preference

Tannins (Drying Sensation)

Questions that measure comfort with bold, drying, astringent sensations:

  • Q1: Coffee - Black coffee requires tannin tolerance similar to red wines
  • Q2: Chocolate - Dark chocolate has tannin-like astringency
  • Q12: Spice Heat - Spice tolerance correlates with tannin acceptance
  • Q14: Fruit Type - Dark berries/plums suggest higher tannin comfort

Body Weight (Fullness/Richness)

Questions that measure preference for bold, rich, full-bodied sensations:

  • Q3: Tea - Strong black tea indicates preference for full-bodied beverages
  • Q7: Main Course - Beef/rich dishes correlate with full-bodied wine preference
  • Q8: Soup - Creamy vs clear soups indicate body preference
  • Q13: Flavor Profile - Earthy/savory flavors suggest bolder wines

Sweetness (Residual Sugar)

Questions that directly measure sweetness preference:

  • Q10: Dessert Importance - Daily dessert need strongly suggests sweet wine preference
  • Q11: Drink Sweetness - Most direct measurement of sweetness preference
  • Q2: Chocolate - White chocolate choice indicates sweet preference
  • Q6: Morning Beverage - Hot chocolate indicates sweetness preference

Flavor Intensity (Boldness)

Questions that measure comfort with bold, intense flavors:

  • Q12: Spice Heat - Maximum spice indicates high flavor intensity tolerance
  • Q9: Pasta Sauce - Hearty/bold sauces indicate intensity preference
  • Q13: Flavor Profile - Rich/savory vs mild indicates flavor intensity preference
  • Q1: Coffee - Strong black coffee indicates boldness preference

Why Questions Don't Directly Ask About Wine

The quiz intentionally avoids direct wine questions because:

  • Most users don't have formal wine education and would struggle with wine terminology
  • Food preferences are more familiar and easier to evaluate honestly
  • Food pairings naturally correlate with wine characteristics (e.g., beef with tannic wines, vinaigrette with acidic wines)
  • The approach works equally well for wine novices and experienced wine drinkers
  • It makes the quiz fun and engaging rather than intimidating
